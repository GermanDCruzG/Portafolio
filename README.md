Portafolio Web – Frontend Developer
Este repositorio contiene mi portafolio personal como desarrollador Frontend, donde presento proyectos desarrollados durante mi proceso de aprendizaje, enfocados en JavaScript, manipulación del DOM, consumo de APIs y diseño responsive.
El objetivo de este portafolio es demostrar mis habilidades técnicas, mi forma de estructurar proyectos y mi evolución como desarrollador web.

Tecnologías utilizadas
HTML5 – Estructura semántica
CSS3 – Estilos personalizados
Bootstrap 5 – Diseño responsive y componentes UI
JavaScript (Vanilla JS) – Lógica, eventos y manipulación del DOM
APIs REST – Consumo de datos externos
LocalStorage – Persistencia de datos en el navegador
Git & GitHub – Control de versiones
